🛠️ Arduino Line-Following Robot Competition
This project showcases an autonomous Arduino-based robot capable of navigating a predefined path and overcoming obstacles. The robot was built and programmed as part of an academic competition, where it ranked among the top performers.

⚙️ Features
Line following with sharp turns

Obstacle detection and avoidance

Reactive speed control for stability

Robust code structure using Arduino C++

🎯 Highlights
One of the top-performing robots in the competition

Focus on efficiency, clarity, and real-world behavior tuning

Developed under tight constraints and real-time evaluation

💻 Technologies
Arduino UNO

IR Sensors

Ultrasonic Distance Sensor

Servo Motor (for steering)

C++ (Arduino IDE)
